# nn-from-scratch

NumPy ile sıfırdan yazılmış basit bir neural network projesi.
Amaç: forward pass, backpropagation ve training mantığını öğrenmek.

## İçerik
- 2 katmanlı MLP (2 -> 3 -> 1)
- Sigmoid aktivasyon
- MSE loss
- Manuel backpropagation
- XOR problemi üzerinde eğitim
